**理由:**

   - ユーザーの制約に適合

2. **Pythonアプリケーションに最適化**
   - FastAPIアプリが簡単に動作
   - 依存関係の管理が容易

3. **無料プランで試せる**
   - 初期費用なし
   - 必要に応じてアップグレード可能

4. **簡単なセットアップ**
   - Webベースのファイルマネージャー
   - コンソールからのコマンド実行

## 📝 PythonAnywhereへのデプロイ手順（概要）

### ステップ1: アカウント作成
1. https://www.pythonanywhere.com にアクセス
2. 無料アカウントを作成

### ステップ2: ファイルのアップロード
1. Filesタブからファイルをアップロード
2. または、Bashコンソールから直接ファイルを作成

### ステップ3: 仮想環境の作成
```bash
mkvirtualenv --python=/usr/bin/python3.11 oreza
pip install -r requirements.txt
```

### ステップ4: Webアプリの設定
1. Web タブから「Add a new web app」
2. Manual configuration を選択
3. WSGI設定ファイルを編集

### ステップ5: 環境変数の設定
- `.env`ファイルまたはPythonAnywhereの環境変数設定

### ステップ6: リロードして起動
- Webアプリをリロード
- URLにアクセスして確認

## ⚠️ 注意事項

### 無料プランの制限

**PythonAnywhere無料プラン:**
- CPU時間: 1日100秒まで
- 外部API: ホワイトリストのみ（OpenAI APIは許可済み）
- ディスク容量: 512MB
- 常時稼働: 可能

**対策:**
- 有料プラン（$5/月）にアップグレード
- または、VPSを検討

### APIキーのセキュリティ

- `.env`ファイルは公開しない
- PythonAnywhereの環境変数機能を使用
- または、Secretsマネージャーを使用

### ドメイン

**無料プラン:**
- `<username>.pythonanywhere.com`

**有料プラン:**
- カスタムドメイン設定可能
